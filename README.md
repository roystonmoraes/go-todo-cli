# Todo CLI (Go + Cobra)

A simple command-line Todo application built with Go, using Cobra for CLI structure and JSON file storage for persistence.

---

## Features

- Add todos
- List todos
- Mark todos as completed
- Delete todos
- Persistent storage using JSON
- Unit-tested business logic
- GitHub Actions CI pipeline

---

## Installation

### Clone the repository

```bash
git clone https://github.com/roystonmoraes/go-todo-cli.git
cd todo-cli
```
